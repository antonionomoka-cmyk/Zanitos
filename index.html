Biblioteca C++ Brasileira Nativa - Versão 1.1

1. Instalação e Configuração

Prepare seu ambiente (Termux ou NetHunter) e baixe a biblioteca oficial:

📥 Baixar Zenitos++ (.zip)

Instalação Global (Terminal)

Após baixar, use os comandos abaixo para instalar a biblioteca no sistema:

unzip zenitos_v1.zip
sudo cp zenitos.h /usr/local/include/
sudo cp libzenitos.a /usr/local/lib/
sudo ldconfig

2. Como Começar?

Crie um arquivo ola.cpp. Como a instalação é global, você pode usar <zenitos.h> em qualquer pasta:

#include <zenitos.h>

principal {
    print(VERDE, "Zenitos++ rodando no sistema!");
    retornar 0;
}

Para compilar usando o binário .a:

g++ ola.cpp -lzenitos -o app && ./app

3. Exemplos Práticos

Nível 1: Interação

texto nome = ler("Digite seu nome: ");
print(AZUL, "Olá,", nome);

Nível 2: Condição (Se/Senão)

inteiro idade = ler_inteiro("Sua idade: ");
se (idade >= 18) {
  print("Acesso Liberado");
} senao {
  print("Menor de idade");
}

Nível 3: Lógica Matemática

decimal nota = ler_decimal("Nota: ");
print("Sua nota convertida é:", nota);
